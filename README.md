# AI Kit Standard

Questa repository contiene un kit documentale standard per guidare il lavoro con agenti AI.
Il contenuto e' generico e va compilato manualmente nel progetto in cui viene usato.

Uso consigliato:
- Copiare la cartella `AI/` nella repository target.
- Compilare i file con placeholder `<<REQUIRED>>` e `<<OPTIONAL>>`.
- Definire gli step operativi in `AI/AI_TASKS.md`.
- Eseguire il lavoro seguendo il Source of Truth dichiarato in `AI/README.md`.
- Aggiornare `AI/KNOWLEDGE.yaml` e `AI/DECISIONS.md` durante l'esecuzione.

Il kit non contiene dati reali di progetto e non deduce struttura o comandi applicativi.

La documentazione completa e' in `AI/README.md`.
